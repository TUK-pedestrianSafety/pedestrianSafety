## 프로젝트명: 교내 보행자 안전을 위한 차량 접근 알림 시스템(Pedestrian Alert System for Campus Vehicles)
- 팀: 소프트웨어 응용설계 2조
- 팀원: 이현직(조장), 송정우, 정승호, 편종윤
  <br>
## 프로젝트 배경 
- 현대인들은 스마트폰을 항상 소지하고 다니며, 걸어다니며 사용하여 사용자가 교내에서 차도에서 위험에 노출되는 경우가 많다.
- 이에, 본 프로젝트는 바닥을 보면서 걷는 위험한 상황에서도 교내 보행자의 안전을 위해 바닥에 센서등을 배치하여 현재 차도가 위험한 상태인지 실시간으로 인지하고 사고를 예방할 수 있도록 하고자 한다.
<br>
## 프로젝트 개요
- 보행자 안전을 위해, 차량 감지 센서를 도로 양쪽끝에 일렬로 두 줄을 설치한다.
- 센서등은 빨강(R), 노랑(Y), 초록(G) 세가지의 표시로 나타내며, 각 센서등은 차량으로 판단되면 차와의 거리에 따라 실시간으로 색이 점등된다.
- 적외선이 감지한 시간을 이용해 거리를 측정한다.
## 운영 규칙
회의 시간은 각자 스케줄에 맞추어 필요한 사안이 있을 시 Discord를 활용한 비대면 회의로 비정기적으로 진행한다.<br>
Discord 및 카카오톡 그룹채팅방을 통해 필요한 소통과 자료 공유 등을 진행한다.
